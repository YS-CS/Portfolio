Console Expense Tracker (Java 21)

Simple console-based expense tracker.

Features:
- Add expenses
- List all expenses
- Show total amount
- Save and load from expenses.csv in the project folder

How to run in Eclipse:
1. Import this folder as an Existing Project into Workspace.
2. Make sure the project uses a JavaSE-21 JRE.
3. Run the Main class (expensetracker.Main) as a Java Application.
